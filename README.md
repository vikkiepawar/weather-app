🌦️ Weather App  
A simple and responsive **Weather Application** built using **HTML, CSS, and JavaScript**.  
This app fetches real-time weather data from the OpenWeatherMap API and displays temperature, humidity, and weather conditions for any city entered by the user.  

🚀 Features  
1. Search weather details for any city
2. Displays temperature, humidity, and weather condition with icons
3. Graceful error handling for invalid inputs or API issues
4. Responsive design for all screen sizes
5. Loading indicator while fetching data
6. Optimized API usage and smooth UI updates  

🛠️ Technologies Used  
 **HTML5** – Structure of the web page  
 **CSS3** – Styling and responsive layout  
 **JavaScript (ES6)** – Fetching API data and DOM manipulation  
 **OpenWeatherMap API** – For real-time weather data  

⚙️ How It Works  
1. User enters the city name in the search input box.  
2. The app sends a **fetch request** to the OpenWeatherMap API.  
3. While fetching data, a **loading indicator** appears.  
4. On success, the app displays the weather details.  
5. On failure (e.g., invalid city or network error), an error message is shown instead.
